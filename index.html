FAQs

Who made this?
It’s made by alexwlchan. If you find it useful, maybe buy me a coffee?
Can you see what videos I’ve liked?
No! This is completely private – everything stays on your phone/computer. I can’t see who you are or what videos you’ve liked on YouTube – that information is never sent to my server.
Beep, boop, I am computer. Can I get my likes in a machine-readable format?
Yes! When you download the HTML file, all the Likes data is stored embedded in a <script> tag. You can extract it and feed it to your own scripts or tools like yt-dlp.
How does it work?
You can “View source” on this web page and see all the code, or look at the GitHub repository.
I love this tool! How can I make sure it doesn’t go away?
You can buy me a coffee which will make me feel happy and keep running my websites, or you can download this web page and get a copy that will run on your computer. (You will need to set up your own Google Cloud credentials.)
I like this tool, but I’d love it even more if it behaved differently! How can I do that?
You can ask me nicely (file a GitHub issue or send me an email), or you can do it yourself! This tool is only three files (HTML, JavaScript, and CSS), so you can download it and then make your own changes to the HTML and JavaScript until it works the way you want.
Something is broken. Can you fix it?
File an issue in the GitHub repository, or send me an email. It’s useful if you tell me what happened when it broke, and what browser you were using.
